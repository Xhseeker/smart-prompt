# “听话的”提示词构建教程

## 项目背景

本教程面向不熟悉LLM的作用机制，但想要构建让LLM“听话”，能够输出符合自己要求内容的提示词的初学者。
本教程旨在帮助初学者快速上手提示词构建，并掌握提示词构建的技巧。

## Roadmap

- [x] “语义压缩”类提示词
- [x] “思维链”类提示词
- [x] “小样本”类提示词
- [x] 元提示词构建提示词
